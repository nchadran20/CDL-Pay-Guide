CDL Class A vs Class B vs Class C: What's the Difference?

Updated: April 2026Read time: 8 min

Before you can get a CDL you need to know which class to get. Most people assume they need a Class A — but that's not always true, and the wrong choice can waste time and money. Here's exactly what each class covers, what jobs it opens up, and which one makes sense for your goals.

Quick Answer for Most People

If you want to drive tractor-trailers (18-wheelers) for OTR, regional, or most trucking careers — you need a Class A CDL. Class B is for straight trucks, school buses, and city vehicles. Class C is for smaller vehicles carrying hazardous materials or 16+ passengers. Class A is the most versatile and highest-paying of the three.

CDL Class A — The Most Common and Highest Paying

A Class A CDL allows you to operate any combination of vehicles with a gross combination weight rating (GCWR) of 26,001 pounds or more — as long as the towed vehicle exceeds 10,000 pounds. In practice this means:

  • Tractor-trailers (18-wheelers, semi-trucks)
  • Flatbed trucks with trailers
  • Refrigerated (reefer) trucks
  • Tanker trucks (with tanker endorsement)
  • Livestock haulers
  • Double and triple trailers (with doubles/triples endorsement)
  • Any vehicle a Class B or C CDL holder can drive

CDL Class B — Straight Trucks and City Vehicles

A Class B CDL covers single vehicles with a GVWR of 26,001 pounds or more — or any such vehicle towing another vehicle not exceeding 10,000 pounds. Class B does not allow you to pull a full trailer. In practice this covers:

  • Straight trucks (box trucks, dump trucks, delivery trucks)
  • School buses (with passenger endorsement)
  • City buses and transit vehicles
  • Garbage trucks
  • Cement mixers
  • Large delivery vehicles (beverage trucks, furniture trucks)

CDL Class C — Smaller Specialty Vehicles

A Class C CDL covers vehicles that don't meet the weight thresholds for Class A or B but are designed to transport 16 or more passengers (including the driver) or placarded hazardous materials. In practice this is a fairly narrow category:

  • Small passenger vans and buses (16+ passengers)
  • Smaller vehicles transporting hazardous materials requiring placards
  • Some airport shuttle vehicles
  • Paratransit vehicles

Class A vs Class B — Which Should You Get?

This is the most common question. Here's the honest answer:

Get Class A if: You want maximum job flexibility, the highest earning potential, or any kind of long-haul career. Class A holders can do everything a Class B holder can do, plus pull full trailers. The salary ceiling is significantly higher.

Get Class B if: You specifically want to drive school buses, city transit, or local delivery trucks and have no interest in tractor-trailer driving. Class B school is shorter and cheaper. But you will be permanently limited to straight trucks — you cannot upgrade to pulling full trailers without going back for Class A training.

Get Class C only if: Your specific job requires it — paratransit work, certain shuttle driving, or small hazmat vehicle operation. Almost no one pursuing a trucking career for income reasons should target Class C.

Can You Upgrade from Class B to Class A?

Yes — but you have to go back to school and pass additional testing. It is not a simple upgrade. You will need to complete the full Class A training including combination vehicle skills, air brakes (if not already endorsed), and a new skills test. Many drivers who start with Class B eventually realize they should have gotten Class A from the beginning. If you're on the fence, get Class A from the start.

Which Class Do Companies Pay to Train For?

Almost all company-paid CDL training programs — Werner, Swift, Schneider, CR England, and others — train exclusively for Class A CDL. If you want a company to pay for your training, you are getting a Class A. See our full guide on companies that pay for CDL training for the complete list.

Bottom Line

For the vast majority of people pursuing a trucking career for income, the answer is Class A. It offers the highest pay, the most job flexibility, and is what most employer-paid training programs provide. Class B makes sense for drivers who specifically want school bus or city transit work and have no interest in long-haul driving. Class C is a niche license for very specific situations.
